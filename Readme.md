**ü§ñ How to Use the Chatbot**

From the home page, click the "Launch Chatbot" button to go to the chat page.

Once on the chat page, you can either:

Click one of the suggestion chips (e.g., "What are common flu symptoms?").

Type your own question into the message box at the bottom.

Press the send button (the blue arrow) or hit Enter on your keyboard.

The chatbot will think for a moment (you'll see a loading animation) and then provide a response in the chat window.
